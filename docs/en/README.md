## Quick Start

Readarr works **out of the box** after building — metadata is served by the community-hosted rreading-glasses instance at `api.bookinfo.pro`. No API keys or configuration needed.

To switch to Open Library:
1. Settings → General → Metadata Source → type `openlibrary` → Save
